Golang 笔记（5）：字符串的更多玩法

 2023-02-21  319 字  golang learning note programming 

字符串与 byte 切片

字符串可以与 []byte 相互转换

str := "abc"  
b := []byte(str)  
s := string(b)

当然这会带来不必要的内存开销，go 为 string 和 []byte 提供了对应功能相同的函数，比如

func main() {  
   str := "abc"  
   fmt.Println(strings.HasPrefix(str, "a"))  
   b := []byte{97, 98, 99}  
   fmt.Println(bytes.HasPrefix(b, []byte{97}))  
}

bytes 包提供了高效处理 []byte 的 Buffer 类型。

func main() {  
   var buf bytes.Buffer  
   buf.WriteByte('[')  
   nums := []int{1, 2, 3}  
   for i, v := range nums {  
      if i > 0 {  
         buf.WriteByte(',')  
      }  
      fmt.Fprintf(&buf, "%d", v)  
   }  
   buf.WriteByte(']')  
   fmt.Println(buf.String())  
}

当写入 ascii 时 WriteByte 已经足够，如果是其他字符比如 utf-8 编码的字符就要用 WriteRune。

字符串与数字

要将整数转换为字符串，可以用 fmt.Sprintf 或strconv.Itoa

num := 100  
str1 := fmt.Sprintf("%d", num)  
str2 := strconv.Itoa(num)

FormatInt 和 FormatUint 可以按不同的进制转换数字。

num := 100  
str := strconv.FormatInt(int64(num), 2)  
fmt.Println(str) //1100100

Sprintf 用于转换字符串更加灵活

num := 100  
str := fmt.Sprintf("x = %b", num)  
fmt.Println(str) //x = 1100100

strconv 包的 Atoi 和 ParseInt 都可以把字符串转换成数字

str := "100"  
num1, err := strconv.Atoi(str)  
num2, err := strconv.ParseInt("123", 10, 64)

注意 ParseInt 的返回值始终是 int64，最后一个参数仅用作校验。
