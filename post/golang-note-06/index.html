数组

长度固定，存储相同数据类型，不常用，一般用切片。
以下是一些数组的声明方式和字面值

var p [3]int //[0,0,0]  
var q [3]int = [3]int{1, 2, 3}  
var r = [3]int{1, 2}         //[1,2,0]，没初始化的都是零值  
var s = [...]int{1, 2, 3, 4} //...代表根据元素个数自动生成数组长度  
t := [...]int{0: 1, 4: 2} //[1 0 0 0 2] 索引:值

数组长度也是数据类型的一部分，也就是说 [3]int 和 [4]int 是不同的类型

var p [3]int 
var q [4]int = p //报错

如果两个数组的元素是可比较的且长度相同，那么这两个数组也是可比较的。当两个数组的元素完全相同时才返回 true。

func main() {  
   a := [2]int{1, 2}  
   b := [...]int{2, 1}  
   c := [...]int{1, 2}  
   fmt.Println(a == b, a == c)  //false true
}

关于数组的传参：
数组在 go 语言是值类型，也就是说默认值传递。

func main() {  
   a := [3]int{1, 2, 3}  
   foo(a)  
   fmt.Println(a) //[1 2 3]  
}  
  
func foo(a [3]int) {  
   a[1] = 0  
}

要想改变数组的值，就要指针传递。

func main() {  
   a := [3]int{1, 2, 3}  
   foo(&a)  
   fmt.Println(a) //[1 0 3]  
}  
  
func foo(a *[3]int) {  
   a[1] = 0  
}

值传递的时候不可避免地拷贝整个数组，很可能造成很大的资源开销，所以尽量使用指针。
