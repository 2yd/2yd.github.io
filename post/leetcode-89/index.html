格雷编码

被格雷编码的生成公式震惊了

以三位格雷编码为例，⊕代表异或

000 ⊕ 000 = 000
001 ⊕ 000 = 001
010 ⊕ 001 = 011
011 ⊕ 001 = 010
100 ⊕ 010 = 110
101 ⊕ 010 = 111
110 ⊕ 011 = 101

可以推导出如下公式： $gray_{i} = i \oplus i»1$

go 代码如下：

func grayCode(n int) []int {  
   var ans []int  
   for i := 0; i < 1<<n; i++ {  
      ans = append(ans, i^i>>1)  
   }  
   return ans  
}

与这道题相关联的 1238. 循环码排列 - 力扣（LeetCode）

  • gray[0] = 0，则 gray[0] ⊕ start = start。
  • gray[i-1]与 gray[i]相差一个二进制位，则都与 start 异或后也只差一个二进制位。
func circularPermutation(n int, start int) []int {  
   var ans []int  
   for i := 0; i < 1<<n; i++ {  
      ans = append(ans, i^i>>1^start)  
   }  
   return ans  
}
