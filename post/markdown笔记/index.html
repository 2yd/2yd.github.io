Markdown笔记

Markdown笔记，用自己的方式记录Markdown语法。

先来一段舒适的音乐

段落

使用空白行将文本进行分割。目前Typora等markdown编辑程序都会自动实现这个过程，不过多赘述。需要注意在Markdown中最好不要使用空格与Tab进行缩进。

换行

在一行的末尾打两个空格并回车即可创建换行。也可以使用html的换行标签<br>进行换行。

简要补充

字体

基本使用：_文本_或*文本*使 文本 变为斜体，__文本__或**文本**使得 文本 变为粗体，~~文本~~为文本加删除线

简要补充：

  1. 斜体和加粗可以同时使用，比如***文本***或**_文本_**都可以使得 文本 变为斜体加粗。更推荐**_文本_**使语法更清晰。
  2. 这些语法当然是可以跨越多个词语的，for example。
  3. 如果文本周围没有空格分隔，比如这是__一句__文本，这种情况下__下划线__会失效，请全部使用星号。

标题

基本使用：标题分为一到六级，一级标题格式# 标题，六级标题格式###### 标题，以此类推。

简要补充：

  1. 也可以在文本下方添加任意数量的=号标识一级标题，-号标识二级标题

  2. 为了兼容考虑需要在#与标题之间使用空格隔开。

  3. 标题文本不能再加粗，但可以加斜体或删除线。

链接

最基本的使用：[链接显示的文本](地址)，比如哔哩哔哩。
